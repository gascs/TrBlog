## 1. 前端性能优化

### 优化项

| 优化内容 | 改进效果 | 实现方式 |

| 图片懒加载 | 减少初始加载时间 | 添加 `loading="lazy"` 属性 |
| 组件记忆化 | 减少不必要的重渲染 | 使用 `memo` 包装组件 |

| React Query 缓存 | 减少重复请求 | 添加 `staleTime` 和 `retry` 配置 |
| 文章卡片组件拆分 | 提高代码可读性 | 拆分为独立的 `PostCard` 组件 |

## 6. 性能对比

### 前端优化前
- 图片立即加载，影响首屏加载速度
- 组件无记忆化，可能导致不必要的重渲染
- 无缓存机制，重复请求相同数据

### 前端优化后
- 图片懒加载，首屏加载速度提升
- 组件记忆化，减少重渲染次数
- 5分钟缓存，减少重复请求

### 后端优化前
- 浏览量增加和查询分开执行，可能导致数据不一致
- 评论无数量限制，可能影响查询性能

### 后端优化后
- 事务处理，确保数据一致性
- 评论数量限制，提高查询性能
